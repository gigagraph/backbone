# Graphics

## Intel

Install drivers for graphics hardware acceleration:

```shell
sudo apt update -y
sudo apt install -y \
  mesa-common-dev \
  libgl1-mesa-dev \
  libgl1-mesa-dri \
  libglapi-mesa \
  libglx-mesa0 \
  libegl-mesa0 \
  libegl1-mesa-dev \
  libosmesa6 \
  libosmesa6-dev \
  libxatracker2 \
  libxatracker-dev \
  mesa-vulkan-drivers \
  mesa-va-drivers \
  mesa-opencl-icd
```
